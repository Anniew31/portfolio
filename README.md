# Annie Weng — Portfolio

Personal portfolio website built with React, Vite, Tailwind CSS, and Framer Motion.

# Tech Stack
- React + Vite
- Tailwind CSS v4
- Framer Motion
- React Router

# Features
- Animated underwater theme with interactive project bubbles
- Project showcase with demo videos
- Fully responsive
